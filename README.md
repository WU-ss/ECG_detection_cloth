# ECG_detection_cloth

基于 **STM32 + MSP430 + ESP8266 + AD8232** 的可穿戴式心电检测衣物项目。项目将心电信号采集、嵌入式数据处理、柔性显示材料、WiFi 物联网通信与衣物集成结合起来，实现面向日常健康监测场景的 ECG 信号采集、显示与云端上传原型系统。

> 本项目仅用于学习、实验和工程原型展示，不作为医疗诊断设备使用。

---

## 项目简介

`ECG_detection_cloth` 是一套可穿戴心电检测衣物软硬件系统。系统通过贴附于人体皮肤的心电传感器采集 ECG 信号，经 STM32 主控端完成 A/D 转换、滤波、心率计算与数据显示，并通过 ESP8266 WiFi 模块将数据上传至物联网平台，实现远程查看和数据可视化。

项目主要包含：

- 人体心电信号（ECG）实时采集；
- 基于 AD8232 的单导联心电采集前端；
- STM32 主控制器进行 ADC 采样、数据处理、显示控制与无线通信；
- MSP430F2001 作为低功耗辅助控制器，用于柔性显示相关控制；
- ESP8266 WiFi 模块实现云端数据上传；
- LCD 屏幕显示心电波形、心率及温湿度等信息；
- 电子胶囊墨水屏用于柔性、轻薄的穿戴式显示尝试；
- PCB / FPC 电路设计与衣物集成。

---

## 项目特点

- **可穿戴式设计**：将 ECG 采集电路与衣物结合，探索智能健康服饰形态。
- **双 MCU 架构**：STM32 负责主控、采样、滤波、显示和联网，MSP430F2001 负责低功耗辅助显示控制。
- **ECG 信号采集与处理**：基于 AD8232 采集微弱心电信号，并通过滤波算法降低干扰。
- **心率检测**：通过 R 波检测和 R-R 间期计算实现心率估计。
- **无线数据上传**：ESP8266 通过 WiFi 接入网络，并基于 MQTT / AT 指令流程上传数据。
- **多显示方式**：支持 LCD 显示心电波形、心率、温湿度信息，并尝试电子胶囊墨水屏柔性显示。
- **完整软硬件工程**：包含 PCB 设计、STM32 程序、MSP430 程序和项目文档。

## ECG 信号处理

心电信号属于低频、微弱且容易受干扰的生物电信号。项目中使用 AD8232 完成心电信号前端采集，并在 STM32 中进行后续处理。

主要处理流程：

1. 通过 AD8232 采集单导联 ECG 信号；
2. 使用 STM32 ADC 对模拟心电信号进行采样；
3. 对采样数据进行 FIR 滤波，降低工频干扰、肌电噪声和基线漂移等影响；
4. 通过动态阈值和斜率特征识别 R 波；
5. 根据相邻 R 波之间的 R-R 间期估算心率；
6. 将心电波形、心率和环境信息显示在 LCD 屏幕上；
7. 通过 ESP8266 将数据上传到云端平台。

## 物联网功能

系统通过 ESP8266 模块连接 WiFi 网络，STM32 使用串口发送 AT 指令控制 ESP8266，并通过 MQTT 协议上传数据。

云端平台可用于：

- 实时查看心率数据；
- 查看温湿度等环境信息；
- 显示 ECG 波形或相关时序数据；
- 构建远程健康监测看板；
- 支持家属或监护人员远程查看数据。

## 快速使用

### 1. 克隆仓库

```bash
git clone git@github.com:WU-ss/ECG_detection_cloth.git
```

如果仓库名为 `ECG_detection_clothing_STM32`，可使用：

```bash
git clone git@github.com:WU-ss/ECG_detection_clothing_STM32.git
```

### 2. 打开 STM32 工程

进入：

```text
Main_MCU_STM32/
```

使用 Keil MDK 打开 STM32 工程文件，编译并生成固件。

### 3. 打开 MSP430 工程

进入：

```text
Screen_MCU_MSP430F2001/
```

使用 TI CCS 打开 MSP430 工程文件，编译并下载到 MSP430F2001。

### 4. 制作硬件

进入：

```text
ECG_detection_clothing_PCB/
```

根据 PCB 文件进行打板、焊接与硬件调试。

### 5. 烧录与运行

1. 将 STM32 固件下载到主控板；
2. 将 MSP430 固件下载到辅助控制板；
3. 正确连接 AD8232、ESP8266、LCD、柔性屏与电源模块；
4. 上电运行；
5. 观察 LCD 显示的心电波形、心率与温湿度信息；
6. 配置 WiFi 与云平台参数后，查看云端数据上传结果。

## 注意事项

1. ECG 信号为微弱生物电信号，硬件调试时需要注意电源噪声、接地、导联连接和人体安全。
2. 本项目仅作为学习与工程原型使用，不具备医疗认证资质，不能用于临床诊断。
3. 使用人体连接电极进行测试时，请确保供电隔离和硬件安全。
4. 若 ECG 波形噪声较大，可检查电极贴合、AD8232 接线、滤波参数和电源纹波。
5. 若 ESP8266 无法上传数据，请检查 WiFi 账号密码、串口波特率、AT 指令流程和云平台配置。
6. 若柔性屏显示异常，请检查 MSP430 程序、I2C 连接、屏幕驱动芯片和引脚映射。
